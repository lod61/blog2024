# Blog2024

基于 Next.js 14 构建的个人博客，支持 Markdown 内容管理、评论系统和响应式设计。

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- Markdown
- Giscus

## 功能特点

- 文章按年份归档
- 标签系统
- 评论功能
- 响应式设计
- 深色模式

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 部署

项目通过 Vercel 自动部署：https://blog2024-lime.vercel.app/
